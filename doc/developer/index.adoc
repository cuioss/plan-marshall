== Workflow discipline

Working in this repository is governed by the hard rules in `CLAUDE.md` (workflow discipline section): script-only `.plan/` access, one Bash command per call, no shell constructs, no workflow improvisation, CI abstraction over direct `gh`/`glab`, build commands resolved through the architecture API, and structured queries before Glob/Grep. Read those rules before your first non-trivial change.
